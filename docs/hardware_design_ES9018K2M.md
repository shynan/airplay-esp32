# ESP32 + ES9018K2M 音频接收器硬件设计

## 一、系统概述

基于ESP32-WROVER-E和ES9018K2M的高端音频接收器，支持：
- AirPlay 2 无线音频
- 蓝牙 A2DP
- 预留 DLNA 扩展
- 预留 OLED/LCD 屏幕接口

---

## 二、核心器件选型

| 器件 | 型号 | 规格 | 数量 |
|------|------|------|------|
| 主控模块 | ESP32-WROVER-E | 8MB PSRAM, 4MB Flash | 1 |
| DAC芯片 | ES9018K2M | 32bit/384kHz, 135dB DR, QFN-28 | 1 |
| 时钟芯片 | Si5351A | 可编程时钟发生器, I2C控制 | 1 |
| 或晶振 | 22.5792MHz / 24.576MHz | 音频专用有源晶振 | 各1 |
| 运放 | OPA1612 / LME49720 | 低噪声双运放 | 1 |
| LDO | TPS7A4700 | 超低噪声 3.3V, 1A | 2 |
| LDO | AMS1117-3.3 | 数字电源 3.3V | 1 |

## 六、时钟方案

ES9018K2M 需要精确的主时钟(MCLK)，提供两种方案：

### 方案A：双晶振方案（简单可靠）

```
                 22.5792MHz                    24.576MHz
              ┌────────────┐               ┌────────────┐
              │ 有源晶振    │               │ 有源晶振    │
              │ (7x5mm)    │               │ (7x5mm)    │
              └─────┬──────┘               └─────┬──────┘
                    │                            │
                    ▼                            ▼
              ┌──────────┐                ┌──────────┐
              │ 74HC4053 │ ◄── GPIO选择 ──│ 74HC4053 │
              │ 模拟开关  │                │ 模拟开关  │
              └────┬─────┘                └──────────┘
                   │
                   ▼
              ES9018 MCLK
```

**晶振选择：**
- 22.5792MHz：用于 44.1kHz / 88.2kHz / 176.4kHz 采样率
- 24.576MHz：用于 48kHz / 96kHz / 192kHz 采样率

### 方案B：Si5351A 可编程时钟（推荐）

```
             ┌─────────────────────────────┐
             │        Si5351A              │
             │                             │
  ESP32 ─────┤ SDA (I2C1)                  │
  GPIO21 ────┤ SCL (I2C1)                  │
             │                             │
             │ CLK0 ───► 22.5792MHz ──────┼──► ES9018 MCLK
             │ CLK1 ───► 24.576MHz ───────┼──► 备用/切换
             │ CLK2 ───► 可配置            │
             │                             │
             │ 25MHz晶振 或 外部参考        │
             └─────────────────────────────┘
```

**Si5351A 优势：**
- I2C 编程，可输出任意频率
- 多路输出，可同时支持两种采样率
- 软件切换，无需模拟开关
- 价格约 ¥8-15

**参考电路：**
```
Si5351A 典型连接:

ESP32 GPIO21 (SDA) ──┬──► Si5351 SDA
                     │
                    4.7K
                     │
                    3.3V

ESP32 GPIO22 (SCL) ──┬──► Si5351 SCL
                     │
                    4.7K
                     │
                    3.3V

Si5351 CLK0 ───────────► ES9018 MCLK (通过100Ω电阻)

Si5351 XA/XB ──────────► 25MHz 晶体 (或外部参考)
```

### 7.2 TPS7A4700 应用电路

```
               TPS7A4700 (超低噪声 LDO, 4.1μVrms)

  5V ──┬── 10μF ──┬─────────────────────────────┬── 10μF ──┬──► 3.3VA
       │         │                             │          │
      ══╪═      ─┴─                           ─┴─       ══╪═
       ═╪═      0.1μF                          0.1μF      ═╪═
       │         │                             │          │
       │         │     ┌───────────────┐       │          │
       └─────────┴─────┤ VIN    VOUT   ├───────┴──────────┘
                       │               │
                       │ EN    PG      │
                       │               │
                       │ GND   NR/SS   ├────────── 0.1μF ──┐
                       │               │                     │
                       └───────────────┘                    GND
                            │
                            GND
```

**关键点：**
- NR/SS 引脚接 0.1μF 电容可进一步降低噪声
- 输入输出电容使用低 ESR 陶瓷电容
- 模拟地和数字地在电源入口单点连接

### 7.3 推荐电源配置

| 电源轨 | 芯片 | 电流 | 用途 |
|--------|------|------|------|
| 3.3VA-1 | TPS7A4700 | 500mA | ES9018 模拟部分、运放 |
| 3.3VA-2 | TPS7A4700 | 100mA | Si5351A 时钟 |
| 3.3VD | AMS1117-3.3 | 1A | ESP32、ES9018数字、屏幕 |

## 十、PCB 布局建议

### 10.1 分层设计

**推荐 4 层 PCB：**

| 层 | 功能 | 说明 |
|---|------|------|
| Top | 信号 + 电源 | ESP32、DAC、运放 |
| Inner 1 | GND | 完整地平面 |
| Inner 2 | 电源 | 3.3VA / 3.3VD 分区 |
| Bottom | 信号 + 电源 | 辅助布线 |

**最小 2 层 PCB：**
- Top: 信号
- Bottom: 地平面 (尽量完整)

### 10.2 布局分区


### 10.3 关键布局规则

1. **ES9018K2M：**
   - EPAD 必须良好接地 (多个过孔到底层地)
   - 每个电源引脚就近放置 100nF 去耦电容
   - AVCC 走线加 π 型滤波 (10Ω + 10μF)

2. **I2S 走线：**
   - BCK、WS、DATA 走线等长 (±5mm)
   - 远离高速数字信号

3. **模拟输出：**
   - 差分走线，等长匹配
   - 远离数字电路和 WiFi 天线

4. **ESP32 天线：**
   - 模块下方不要铺铜
   - 保持至少 15mm 净空区

5. **地分割：**
   - 数字地 (DGND) 和模拟地 (AGND) 分区
   - 在电源入口处单点连接 (0Ω 电阻或磁珠)

## 十三、软件配置要点

### 13.1 ESP32 I2S 配置

```c
// i2s_config.h
#define I2S_BCK_IO      15
#define I2S_WS_IO       14
#define I2S_DO_IO       13
#define I2S_MCLK_IO     16  // 可选，ES9018使用外部MCLK

// ES9018K2M 支持 32-bit 数据
i2s_config_t i2s_config = {
    .mode = I2S_MODE_MASTER | I2S_MODE_TX,
    .sample_rate = 44100,
    .bits_per_sample = I2S_BITS_PER_SAMPLE_32BIT,
    .channel_format = I2S_CHANNEL_FMT_RIGHT_LEFT,
    .communication_format = I2S_COMM_FORMAT_STAND_I2S,
    .dma_buf_count = 8,
    .dma_buf_len = 256,
    .use_apll = true,  // 使用 APLL 获得更精确的时钟
    .tx_desc_auto_clear = true,
};
```

### 13.2 Si5351A 时钟配置

```c
// si5351.h
#define SI5351_I2C_ADDR     0x60

// 输出频率配置
// 44.1kHz 系列: MCLK = 512 * fs = 22.5792MHz
// 48kHz 系列:   MCLK = 512 * fs = 24.576MHz

void si5351_set_freq(uint32_t freq_hz, uint8_t clk_output) {
    // 根据采样率设置时钟频率
    if (freq_hz == 44100 || freq_hz == 88200 || freq_hz == 176400) {
        si5351_set_clk(clk_output, 22579200);  // 22.5792MHz
    } else {
        si5351_set_clk(clk_output, 24576000);  // 24.576MHz
    }
}
```

### 13.3 ES9018K2M I2C 寄存器

```c
// es9018.h
#define ES9018_I2C_ADDR     0x48  // 或 0x49，取决于 ADDR 引脚

// 关键寄存器
#define ES9018_REG_00       0x00  // 系统设置
#define ES9018_REG_01       0x01  // 输入设置
#define ES9018_REG_02       0x02  // 滤波器设置
#define ES9018_REG_03       0x03  // 时钟管理
#define ES9018_REG_04       0x04  // 自动静音
#define ES9018_REG_05       0x05  // 音量控制

// 初始化序列
void es9018_init(void) {
    // 设置 I2S 输入格式
    es9018_write(0x01, 0x00);  // I2S, 32-bit

    // 设置滤波器
    es9018_write(0x02, 0x04);  // 最小相位快速滚降

    // 设置时钟
    es9018_write(0x03, 0x00);  // 自动检测 MCLK/LRCK 比率

    // 设置音量 (0dB)
    es9018_write(0x05, 0x00);
}
```
