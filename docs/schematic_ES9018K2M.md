# ESP32 + ES9018K2M 原理图连接表

## 一、完整引脚连接表

### 1.1 ESP32-WROVER-E 引脚分配

```
ESP32-WROVER-E (30Pin 模块)
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  Pin 1  (GND)      ─────► GND                                  │
│  Pin 2  (3V3)      ◄───── 3.3VD (数字电源)                      │
│  Pin 3  (EN)       ─────► EN (复位，接10K上拉+0.1μF到GND)       │
│  Pin 4  (GPIO36)   ◄───── 按键 VOL- (需外部10K上拉)             │
│  Pin 5  (GPIO39)   ◄───── 按键 VOL+ (需外部10K上拉)             │
│  Pin 6  (GPIO34)   ◄───── 按键 PLAY (需外部10K上拉)             │
│  Pin 7  (GPIO35)   ◄───── 耳机检测 JACK_DET (需外部10K上拉)     │
│  Pin 8  (GPIO15)   ─────► ES9018 BCK (位时钟)                   │
│  Pin 9  (GPIO2)    ◄────► ES9018 SDA (I2C数据，需4.7K上拉)      │
│  Pin 10 (GPIO0)    ─────► BOOT按键 (需10K上拉)                  │
│  Pin 11 (GPIO4)    ─────► ES9018 SCL (I2C时钟，需4.7K上拉)      │
│  Pin 12 (GPIO16)   ─────► (可选I2S_MCLK，ES9018不用)            │
│  Pin 13 (GPIO17)   ─────► 预留GPIO                              │
│  Pin 14 (GPIO5)    ─────► ES9018 RESET                          │
│  Pin 15 (GPIO18)   ─────► 屏幕DC / W5500 SPI_CLK                │
│  Pin 16 (GPIO19)   ─────► 屏幕RES / W5500 SPI_MISO              │
│  Pin 17 (GPIO21)   ◄────► Si5351 SDA (I2C1数据，需4.7K上拉)     │
│  Pin 18 (GPIO3)    ◄───── UART_RX (下载/调试)                   │
│  Pin 19 (GPIO1)    ─────► UART_TX (下载/调试)                   │
│  Pin 20 (GPIO22)   ─────► Si5351 SCL (I2C1时钟，需4.7K上拉)     │
│  Pin 21 (GPIO23)   ─────► 屏幕MOSI / W5500 SPI_MOSI             │
│  Pin 22 (GND)      ─────► GND                                  │
│  Pin 23 (GPIO13)   ─────► ES9018 SDIN (音频数据)                │
│  Pin 24 (GPIO12)   ─────► 预留GPIO (注意:启动时需低电平)        │
│  Pin 25 (GPIO14)   ─────► ES9018 LRCK (字选择)                  │
│  Pin 26 (GPIO27)   ─────► ES9018 MUTE (静音控制)                │
│  Pin 27 (GPIO26)   ─────► 屏幕SCK                               │
│  Pin 28 (GPIO25)   ─────► 屏幕CS                                │
│  Pin 29 (GPIO32)   ─────► WS2812 RGB LED                        │
│  Pin 30 (GPIO33)   ─────► 触摸屏T_CS (可选)                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 模拟电源 (3.3VA) - 超低噪声

```
                    模拟电源电路 (TPS7A4700)
                    ─────────────────────────

                              TPS7A4700
                         ┌─────────────────┐
                         │                 │
            5V_MAIN ─────┤ VIN         VOUT├────┬────┬────► 3.3VA
                         │                 │    │    │
            5V_MAIN ─────┤ EN           NR ├────┘    │
                         │                 │    ═══  │
                         │ GND             │   0.1μF │
                         │                 │    │    │
                         └────────┬────────┘   GND   ═══ 10μF
                                  │                  │
                                 GND                 GND

    关键点:
    - NR引脚接0.1μF可降噪至 4.1μVrms
    - 输出电容使用低ESR陶瓷电容

    额外LC滤波 (可选，进一步降噪):

    3.3VA ──┬── 10μH ──┬──► 3.3VA_CLEAN
            │          │
           ═══        ═══
          100μF       10μF
            │          │
           GND        GND

    DAC AVCC 引脚滤波:

    3.3VA ── 10Ω ──┬──► ES9018 AVCC
                   │
                  ═══ 10μF
                   │
                  GND
```

    GND              ───────► GND

    注意: 与屏幕 SPI 引脚复用，需要不同 CS


    网络变压器 (如果使用RJ45带变压器的模块则不需要):
    ─────────────────────────────────────────────────────────────

    W5500 模块通常已集成网络变压器，直接连接 RJ45 即可
```

## 九、地线设计

```
                    地线分割方案
                    ────────────

    ┌─────────────────────────────────────────────────────────────┐
    │                            PCB                              │
    │                                                             │
    │   ┌─────────────────────────┐   ┌─────────────────────────┐│
    │   │                         │   │                         ││
    │   │       DGND              │   │        AGND            ││
    │   │    (数字地区域)          │   │     (模拟地区域)        ││
    │   │                         │   │                         ││
    │   │   ESP32                 │   │   ES9018               ││
    │   │   Si5351A               │   │   OPA1612              ││
    │   │   屏幕                  │   │   输出端子             ││
    │   │                         │   │                         ││
    │   │                         │   │                         ││
    │   └────────────┬────────────┘   └────────────┬────────────┘│
    │                │                             │             │
    │                │                             │             │
    │                └────────────┬────────────────┘             │
    │                             │                              │
    │                             ▼                              │
    │                     ┌──────────────┐                       │
    │                     │   单点连接    │                       │
    │                     │              │                       │
    │                     │   0Ω 电阻    │                       │
    │                     │   或 磁珠    │                       │
    │                     │              │                       │
    │                     └──────┬───────┘                       │
    │                            │                               │
    │                            ▼                               │
    │                      电源入口 GND                           │
    │                                                            │
    └─────────────────────────────────────────────────────────────┘


    接地要点:
    ─────────────────────────────────────────────────────────────
    1. DGND 和 AGND 在物理上分开，仅通过一点连接
    2. 连接点选择在电源入口处
    3. 使用 0Ω 电阻或磁珠连接，便于调试
    4. AGND 区域避免数字信号穿越
    5. ES9018 EPAD 必须可靠连接到 AGND
