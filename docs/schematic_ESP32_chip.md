## 四、Strapping 引脚电路

### 4.1 GPIO0 - 启动模式选择

```
                    GPIO0 Strapping 电路
                    ────────────────────

         3.3V
           │
          10K (上拉电阻，确保正常运行模式)
           │
    GPIO0 ─┼────────────────────────────────────┐
           │                                    │
          ═══ 100pF (消抖)                      │
           │                                    │
          GND                                   │
                                                │
           │                                    │
          按键 ─────────────────────────────────┘
           │
          GND

    工作模式:
    ───────────────────────────────────────────────────────
    | GPIO0 状态  | 模式               | 说明              |
    |-------------|-------------------|-------------------|
    | HIGH (默认) | 正常运行模式       | 从Flash启动       |
    | LOW (按键)  | 下载模式           | 进入Bootloader    |
    ───────────────────────────────────────────────────────
```

### 4.2 GPIO12 - Flash 电压选择 (关键!)

```
                    GPIO12 Strapping 电路
                    ────────────────────────

         3.3V
           │
          NC (不连接!)
           │
    GPIO12 ─┼────────────────────────────────────► 必须为 LOW!
           │
          10K (下拉电阻 - 必须焊接!)
           │
          GND

    ⚠️⚠️⚠️ 重要警告 ⚠️⚠️⚠️
    ───────────────────────────────────────────────────────
    GPIO12 决定 Flash 工作电压:

    | GPIO12 状态 | Flash 电压 | 说明                    |
    |-------------|-----------|-------------------------|
    | LOW (默认)  | 3.3V      | 大多数Flash芯片 (正确)   |
    | HIGH        | 1.8V      | 特殊Flash芯片 (错误!)    |

    如果 GPIO12 未下拉，ESP32 会尝试以 1.8V 模式访问 Flash，
    导致无法启动！这是裸芯片设计最常见的错误！
    ───────────────────────────────────────────────────────
```

## 十三、PCB 布局检查清单

```
                    PCB 设计检查清单
                    ─────────────────

    □ 电源部分
      □ 每个ESP32电源引脚有100nF去耦电容
      □ Flash/PSRAM电源有10μF大电容
      □ 模拟电源与数字电源分离
      □ 地平面完整，无分割穿越

    □ Flash/PSRAM
      □ Flash距离ESP32 < 30mm
      □ SPI走线下方有完整地平面
      □ 时钟线最短，有地线保护
      □ PSRAM与Flash总线走线匹配

    □ Strapping 引脚
      □ GPIO12 必须有10K下拉电阻
      □ GPIO0 有10K上拉电阻
      □ GPIO15 有10K上拉电阻
      □ 启动时各Strapping状态正确

    □ RF 天线
      □ 天线净空区 ≥ 15×30mm
      □ 所有层净空
      □ 匹配网络靠近天线
      □ 地平面完整

    □ 模拟电路
      □ DAC远离数字电路
      □ 模拟地与数字地单点连接
      □ 音频输出走线差分匹配

    □ 用户界面
      □ GPIO34-39 有外部上拉
      □ 按键消抖电容
      □ 耳机检测上拉

    □ 其他
      □ USB ESD保护
      □ 测试点预留
      □ LED限流电阻
```
